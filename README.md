# Tirakan Essential Foundry System

This is a Foundry VTT System for the dark fantasy role playing game *Tirakans Reiche* in the *Tirakan Essential* variant.

## System

The system provides basic features like character management, dice rolling and character imports.

- Character and NSC actor types.
- Actor sheets with direct roll buttons.
- W100 attribute rolls with target values, result levels, skill-rank consequences, and chat output.
- Wounds, burden, omen, arkana, favor, conditions, and derived values.
- Importer under system settings for a character hash from the chracter creation site [tirakan.de](https://tirakan.de).
- Local English and German localization.
